🚀 바로 시작하는 방법
================================================================================

1. 환경 설정:
   cd /home/user/webapp
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt

2. 환경 변수 설정:
   cp .env.example .env
   nano .env

3. 테스트 실행:
   python test_wconcept_scraper.py

4. 개발 시작:
   - 실제 페이지 구조 분석 (개발자 도구)
   - CSS 셀렉터 확인
   - 크롤러 프로토타입 구현
